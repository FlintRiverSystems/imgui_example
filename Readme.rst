================================================================================
WASM build
================================================================================

Install emsdk::

   cd $FRSTPSRC
   git clone https://github.com/emscripten-core/emsdk
   cd emsdk
   ./emsdk install latest
   ./emsdk activate latest

Now run ``./build_wasm.sh`` in the imgui_example folder
